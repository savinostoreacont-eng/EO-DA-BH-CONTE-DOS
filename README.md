# 🎬 EO DA BH CONTEÚDOS

Um reprodutor de vídeos e áudios simples e direto, sem complicações!

## 📁 Estrutura

```
eo-da-bh/
├── index.html          (Arquivo principal - abra este no navegador)
├── videos/             (Pasta com seus vídeos e áudios)
│   ├── seu-video.mp4
│   ├── outro-video.mp4
│   └── audio.mp3
└── README.md
```

## 🚀 Como Usar

1. **Adicione seus vídeos/áudios**: Coloque seus arquivos MP4, MP3, WebM, OGG, MOV ou AVI na pasta `videos/`

2. **Abra o arquivo**: Abra `index.html` no seu navegador (pode ser local ou em um servidor)

3. **Pronto!** Os vídeos aparecerão automaticamente na biblioteca

## 📝 Formatos Suportados

- **Vídeos**: MP4, WebM, OGG, MOV, AVI
- **Áudios**: MP3, OGG

## ✨ Funcionalidades

✅ Reprodutor com controles (play, pausa, volume, fullscreen)  
✅ Biblioteca visual com cards dos vídeos  
✅ Detecção automática de novos arquivos  
✅ Interface responsiva (funciona em desktop, tablet e mobile)  
✅ Suporte a vídeos e áudios  
✅ Design moderno com gradiente roxo  

## 💡 Dicas

- Os vídeos são carregados automaticamente a cada 5 segundos
- Você pode clicar nos cards para reproduzir os vídeos
- Use o dropdown para selecionar manualmente
- Funciona 100% local - sem necessidade de servidor

## 📦 Hospedagem

Para hospedar online:
- Coloque a pasta `eo-da-bh` em um servidor web
- Certifique-se de que a pasta `videos/` tem permissão de leitura
- Acesse via URL do seu domínio
